# ARCHIVAGE_UNILU

Portail d'archivage des résultats académiques de l'Université de Lubumbashi (UNILU).

## Aperçu
Ce système permet de :
- Afficher les résultats académiques sous forme de dashboard moderne.
- Effectuer des recherches d'étudiants par matricule ou par nom.
- Générer des documents officiels (PDF/Impression) à partir des données extraites.

## Technologies
- HTML5 / CSS3 / JavaScript (Vanilla)
- Python (pour l'extraction des données Excel vers JSON)

## Installation & Usage
1. Cloner le dépôt.
2. Ouvrir `index.html` pour accéder au portail.
3. Les données sont chargées dynamiquement depuis `students_results.json`.
